# Gollection

Gollection provides `Generic Data Structure`.
 Queue, Stack and List like C# in go.



## Requirement
_GO 1.21 or higher_

## Support Structure
- Set
- Queue
- Stack
- Priority Queue
- B Tree
- AVL Tree

#### _`sync` is implemented for thread safe_
- Sync Map
- Sync Set
- Sync Queue
- Sync Stack
- Sync Priority Queue
- Sync B Tree
- Sync AVL Tree

## Import
```go
import "github.com/yanun0323/gollection/v2"
```
